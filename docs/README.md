# Documentação do SmokeGuard

Este diretório é destinado à documentação complementar do projeto, incluindo:

- diagramas da arquitetura;
- esquemas de ligação dos sensores e LEDs;
- instruções de instalação;
- registros de testes;
- imagens do protótipo;
- documentação operacional do sistema.

Projeto desenvolvido no IFSP — Campus Catanduva.
